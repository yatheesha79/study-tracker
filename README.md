# 📚 Smart Study Planner

Smart Study Planner is a simple and interactive web application designed to help students manage their study schedule effectively.

## 🚀 Features
- Add subjects with exam dates
- Automatically generates daily study plan
- Track completed tasks with "Done" functionality
- Prevents duplicate subjects
- Expandable subject-wise task view
- Automatically removes subject after completion (with confirmation)
- Data persistence using localStorage (no data loss after refresh)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Browser Local Storage

## 💡 How It Works
Users can enter subjects along with exam dates. The system calculates the number of days remaining and generates a study plan accordingly. Tasks can be marked as completed, and once all tasks are finished, the subject is removed after user confirmation.

## 🎯 Purpose
This project was built to improve productivity and help students stay consistent with their study routine.
